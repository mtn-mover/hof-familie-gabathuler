# Hof Familie Gabathuler

Offizielle Website für den Bauernhof Familie Gabathuler in Fläsch, Graubünden.

## Über das Projekt

Diese Website präsentiert den landwirtschaftlichen Betrieb der Familie Gabathuler-Risch mit Fokus auf:
- **Mutterkuhhaltung**: Artgerechte Tierhaltung in den Bündner Bergen
- **Hofladen**: Direktverkauf von frischem Fleisch und Wurstwaren
- **Bio Suisse**: Zertifizierte biologische Landwirtschaft

## Technologie-Stack

- **Framework**: Next.js 14
- **Sprache**: TypeScript
- **Styling**: Tailwind CSS
- **Animationen**: Framer Motion
- **Deployment**: Vercel

## Design

Die Website ist inspiriert von:
- [kuppelwieser.bio](https://kuppelwieser.bio) - Modernes, luftiges Layout
- [coms.ch](https://coms.ch) - Warme, natürliche Farbpalette

### Farbschema

| Farbe | Hex | Verwendung |
|-------|-----|------------|
| Primär | `#8f7558` | Brauntöne für Text und Akzente |
| Sekundär | `#418b68` | Grün für Buttons und Highlights |
| Akzent | `#ed900f` | Gold für Call-to-Actions |
| Hintergrund | `#f5f3f0` | Beige/Sand für Backgrounds |

### Typografie

- **Überschriften**: Merriweather (Serif)
- **Fliesstext**: Inter (Sans-Serif)

## Installation

```bash
# Dependencies installieren
npm install

# Entwicklungsserver starten
npm run dev

# Produktions-Build erstellen
npm run build

# Produktionsserver starten
npm start
```
